# INI片段总目录与说明文档

## 简介

本项目是红色警戒2 INI片段精炼厂，汇集了各种常用红警2（红色警戒2原版及其模组）的INI配置片段，可以在地图制作过程中快速插入使用。

## 许可证

MIT

***

## 使用方法

这些INI片段可在红警地图编辑器中使用：

1. 打开地图编辑器
2. 进入"INI编辑器"
3. 选择"插入其他INI文件内容"
4. 选择需要的INI片段文件插入到地图中

***

| 文件名             | 说明                 |

| 文件名          | 说明                       |

| ---------- | ------------------------------------------------------------------------ |
| 常规缺陷修复.ini | 修复官方遗留的缺陷，包括引起弹窗报错的bug、V3/无畏/雷鸣飞弹3星动画丢失、鲍里斯3星米格数量异常等。推荐新建地图时增加导入，免除后顾之忧。 |
