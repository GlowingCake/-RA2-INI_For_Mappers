# 红警INI片段百宝箱

作者：cake1 

作者GitHub：[GlowingCake](https://github.com/GlowingCake)

## 简介

这个仓库汇集了各种常用红警2（红色警戒2原版及其模组）的INI配置片段，可以在地图制作过程中快速插入使用，大大节省地图作者的时间和精力。

## 使用方法

这些INI片段可在红警地图编辑器中使用：

1. 打开地图编辑器
2. 进入"INI编辑器"
3. 选择"插入其他INI文件内容"
4. 选择需要的INI片段文件插入到地图中

## INI片段一览

### 根目录片段

- **倒计时剩余时间警告语音.ini** - 提供游戏时间剩余警告语音触发
- **禁止在盟友基地建造东西.ini** - 禁止玩家在盟友基地建造建筑
- **矿很难采完，钱不变吸的速度变快.ini** - 修改矿石属性
- **生死斗模式YR转RA2（禁用YR科技）.ini** - 将YR模式转换为RA2模式，禁用YR新单位
- **苏盟作战转生死斗(生死斗模式打乱科技点数，这时可考虑伪生死斗）.ini** - 从结果上转换游戏模式并调整科技树
- **特色单位阵营间共享.ini** - 实现特色单位在不同子阵营间共享
- **心灵终结时静光照.ini** - 添加特殊的光照效果，可开关控制

### 常用片段（每张地图都用得上）

- **基地绑间谍卫星效果（开全图，但失去黑全图）.ini** - 为基地添加间谍卫星效果
- **建造上限99.ini** - 提高建筑队列上限至99
- **禁船厂.ini** - 禁用所有船厂建造
- **禁止初始部队（无论玩家如何设置，都不刷初始兵和坦克）.ini** - 完全禁用初始部队
- **开局单次更改所属，位置A到H都有对应触发.ini** - 开局时更改玩家所属
- **玛雅金字塔加强debug（测图专用，挡电脑进攻）.ini** - 增强玛雅金字塔用于测试
- **清除黑幕（开局所有人开全图）.ini** - 开局清除所有战争迷雾
- **虚拟金钱建筑（注册获得时给钱的空建筑，按需给予玩家即可）.ini** - 添加虚拟金钱建筑

### 光照设置

- **沙漠-暗橙色.ini** - 沙漠地图暗橙色光照效果
- **沙漠-阴冷青色.ini** - 沙漠地图阴冷青色光照效果

### 金钱判断难度

- **开局资金判断难度，设置局部变量.ini** - 根据开局资金判断游戏难度

### AI图（PVE）通用

- **开局改所属A到H延迟3秒（防AI展开基地拉兵）.ini** - 延迟3秒更改AI所属
- **开局改所属A到H延迟5分钟（用于给AI启用额外建筑，太早玩家打不了）.ini** - 延迟5分钟更改AI所属
- **科技电厂增强（给AI免除后顾之忧）.ini** - 增强科技电厂属性
- **五倍矿棒效率（解决人多矿不够问题）.ini** - 提高矿石生长速度
- **主战坦克造速1.0（取消YR造速惩罚）.ini** - 取消坦克建造速度惩罚
