# 红色警戒2 INI片段精炼厂

创建者：[cake1](https://github.com/GlowingCake)

GitHub仓库：[-RA2-INI\_For\_Mappers](https://github.com/GlowingCake/-RA2-INI_For_Mappers)

## 简介

这个仓库汇集了各种常用红警2（红色警戒2原版及其模组）的INI配置片段，可以在地图制作过程中快速插入使用，极大的节省地图作者的时间和精力。

## 目录结构

```
-RA2-INI_For_Mappers/
├── 通用/          # 每张地图都可能用到的通用配置片段
├── AI-PVE/        # PVE模式AI相关配置
├── 任务地图/      # 任务地图制作相关的模板和配置
├── 光照/          # 光照效果配置文件
├── 游戏规则/      # 游戏规则修改配置
├── 总目录和说明文档.md  # 完整的目录索引和文件说明
└── LICENSE
```
