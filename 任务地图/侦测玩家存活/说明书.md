# 侦测玩家存活 说明书

## 使用方法
需要在地图外放一个666号路径点。

## 功能说明
所有玩家(或AI)存活状态对应局部变量90至97，例如：
- 90对应玩家A
- 玩家存活：该变量为1
- 玩家被击败或该位置无玩家：变量为0

## 占用资源

| 类型 | 占用编号 |
|------|---------|
| 局部变量 | 90~97 |
| [VehicleTypes] | 660~667 |
| 地编可见内容 | 开头标注[TestPlayer] |
| ini相关内容 | 开头标注CATP |

## 注意事项

游戏开始时所有相关变量都是0，游戏开始时触发会把存活玩家(或AI)变量0改为1，花费时间不到1秒。
